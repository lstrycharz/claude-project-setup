# Strategic Initiative — Planning Rules

Rules for larger planning projects that produce why/what/how documentation (PRD, Spec, Work Plan). The aim is to make decisions explicit, milestones measurable, and the planning artifacts useful as living documents rather than write-once-and-forget.
